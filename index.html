```

Коротко

Бот принимает текстовую тему и возвращает готовый продающий контент, используя GPT-4o mini через OpenAI API.

Как запустить (локально / VPS / PythonAnywhere)

  1. Клонировать репозиторий: git clone https://github.com/твой-ник/tg-bot-generator.git
  2. Перейти в папку: cd tg-bot-generator
  3. Установить зависимости: pip install -r requirements.txt
  4. Задать переменные окружения (пример):
export BOT_TOKEN="тут_твой_telegram_token"\nexport OPENAI_API_KEY="sk-..."
  1. Запустить: python3 main.py (или через процесс-менеджер / systemd / Procfile)

Тарифы и тест

По умолчанию в репозитории указаны пакеты для запуска MVP. OpenAI-расходы зависят от количества генераций — контролируй через панель OpenAI.

Полезные команды

git pull origin main\npip install -r requirements.txt\npython3 main.py
v1.0 — MVP
Запустить бота
Репозиторий на GitHub OpenAI API Keys Запуск на PythonAnywhere VPS (Timeweb)

Переменные окружения

  • BOT_TOKEN — токен от @BotFather
  • OPENAI_API_KEY — ключ OpenAI

Примечание

Если ты хочешь, чтобы хостинг сам управлял запуском — удали из main.py блок с запуском (async def main / asyncio.run(...)) и добавь Procfile или systemd unit.

Если нужно — я помогу: обновлю README, добавлю Procfile или подготовлю systemd unit для VPS.
```
